_**Project from Stanford's CS106A class, made using Eclipse.**_  
# To run from command line:

## Windows:
Compile: `javac -cp ".;lib/sqlite-jdbc-3.18.0.jar;acm.jar" *.java`  
Run: `java -cp ".;lib/sqlite-jdbc-3.18.0.jar;acm.jar" FacePamphlet`

## Mac:
Compile: `javac -cp ".:lib/sqlite-jdbc-3.18.0.jar:acm.jar" *.java`  
Run: `java -cp ".:lib/sqlite-jdbc-3.18.0.jar:acm.jar" FacePamphlet`
